# Snake Game Web App

เกมงูที่สร้างด้วย Python และรันในเว็บเบราว์เซอร์โดยใช้ PyScript

## วิธีการเล่น
- ใช้ปุ่มลูกศร (↑ ↓ ← →) เพื่อควบคุมทิศทางการเคลื่อนที่ของงู
- งูจะเคลื่อนที่อัตโนมัติ กินอาหารสีแดงเพื่อเพิ่มคะแนนและความยาว
- หลีกเลี่ยงการชนขอบหน้าจอหรือตัวเอง
- จบเกมเมื่อชนขอบหรือตัวเอง

## วิธีรันบน GitHub Pages
1. สร้าง repository ใหม่บน GitHub
2. Upload โฟลเดอร์ `snake-game` นี้ไปยัง repository
3. ไปที่ Settings > Pages ใน repository
4. เลือก branch (ปกติ main) และโฟลเดอร์ / (root)
5. บันทึก และรอให้ GitHub สร้าง URL สำหรับหน้าเว็บ
6. เปิด URL ที่ได้เพื่อเล่นเกม

## เทคโนโลยีที่ใช้
- **PyScript**: ใช้รัน Python code ในเบราว์เซอร์
- **Turtle Graphics**: สำหรับวาดและควบคุมกราฟิกเกม
- **HTML/CSS**: สำหรับโครงสร้างและสไตล์ของหน้าเว็บ

## หมายเหตุ
เกมนี้รันได้ในเบราว์เซอร์ที่รองรับ WebAssembly และ PyScript หากโหลดช้าในครั้งแรก ให้รอสักครู่
